# advik_143bot

A more featured TG bot I made, features are added regularl!

## Prerequisites

You can use python 3.9, or pycharm or any other IDEs.
Required modules installation:
- requests
- pyTelegrambotAPI
- Pillow
- qrcode

```
pip install requests
```
```
pip install pyTelegrambotAPI
```
```
pip install Pillow
```
```
pip install qrcode
```
