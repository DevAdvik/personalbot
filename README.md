# DevAdvikBot

A more featured TG bot I made!

## Prerequisites

You can use python 3.9, or pycharm or any other IDEs.
Required modules installation:
- requests
- pyTelegrambotAPI
- Pillow
- qrcode
- pyshorteners

```
pip install requests
```
```
pip install pyTelegrambotAPI
```
```
pip install Pillow
```
```
pip install qrcode
```
```
pip install pyshorteners
```
